# Certificate Page (React + Vite + Tailwind)

A simple one-page site to showcase certificates with search, category filters, and a modal preview.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy (Netlify)
- Build command: `npm run build`
- Publish directory: `dist`

## Add your certificate images
Place images in `public/certs/` and update `src/data/certificates.js`.
